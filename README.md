# Momentum Tracker

一个集成了Firebase的日常任务和记录跟踪应用。

## 功能特性

- 📝 日常记录管理
- ✅ 任务跟踪和完成状态
- 📅 日历视图和进度统计
- 🔐 Google账户登录
- ☁️ 云端数据同步
- 📱 响应式设计

## 技术栈

- HTML5 + CSS3 + JavaScript
- Firebase Authentication
- Firebase Firestore
- Google OAuth 2.0

## 部署

本项目已配置为Vercel部署，可以直接部署到Vercel平台。

## 本地开发

```bash
python -m http.server 8000
```

然后访问 `http://localhost:8000`
